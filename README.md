UnseenConnect — приложение для **Android** для удобного управления прокси-серверами на базе ядра Xray. Простой интерфейс, импорт подписки и быстрое подключение к нужному серверу.

---

## Установка

1. Скачайте **UnseenConnect.apk** по кнопке выше.
2. Откройте файл — Android попросит разрешение на установку из неизвестных источников.
3. Разрешите и нажмите **Установить**.
4. Вставьте ссылку на подписку или конфиг vless://.
5. Выберите сервер и нажмите **Подключить**.

> Если установка блокируется — перейдите в **Настройки → Безопасность → Установка неизвестных приложений** и разрешите для вашего браузера/файлового менеджера.

---

## Возможности

* Импорт подписки и конфигов в один клик
* Системный прокси и полноценный VPN-режим (TUN)
* Список серверов с пингом и выбором лучшего узла
* Автообновление подписки
* Автоматическое обновление приложения
* Тёмная тема, поддержка нескольких языков
* Только портретная ориентация

## Поддерживаемые протоколы

* **VLESS (Reality)**
* **VMess**
* **Trojan**
* **Shadowsocks**
* **SOCKS**

---

UnseenConnect не собирает ваши данные — всё хранится только на вашем устройстве.

Приложение **не продаёт VPN-подписки**. Серверы вы подключаете самостоятельно у своего провайдера. Используйте сервис в соответствии с законодательством вашей страны.
